# Wine App React (WIP)

Light weight react redux app that searches for wine across various lot codes and provides break downs.

Uses [Create React App]("https://create-react-app.dev")

## Technology used
* React
* Redux
* Thunks
* Styled components
* Reselect
* Redux Persist

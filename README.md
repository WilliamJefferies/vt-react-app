# VT-React-App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

This is a simple react app that searches for wine across various lotcodes and provides breakdowns.

## Technology used
* React
* Redux
* Thunks
* Styled components
* Reselect
* Redux Persist
